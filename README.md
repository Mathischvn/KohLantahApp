## Lancer le projet

Pour installer, les dépendances du projet:
```bash
npm install
```

Pour lancer le projet:   
```bash
npm run dev
```
Le projet sera accessible sur [http://localhost:3000/](http://localhost:3000/)

## Build le projet

Pour build le projet:
```bash
npm run build
```

## Pour récuperer les dernières mises à jour
```bash
git pull
```
